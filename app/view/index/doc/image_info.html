# 请求信息

功能 接口
请求方式 POST\GET
URL {:url('api/imageInfo')->suffix('')->domain(true)}

请求参数

参数名称 类型 是否必须 说明
token String 是 用户的秘钥
signatures String 是 图片唯一标识

响应参数

参数名称 类型 实例值 说明
code Number 200 状态码，成功返回200，失败返回400
msg String success 提示信息
