# 请求信息

功能 接口
请求方式 POST\GET
URL {:url('api/imageUpdate')->suffix('')->domain(true)}

请求参数

参数名称 类型 是否必须 说明
token String 是 用户的秘钥
signatures String 是 图片唯一标识
publicCloud String 是 公有云CDN，上传的图床类型（this、ali、huluxia等等），支持字符串传入（英文逗号分隔，例如：ali,huluxia）
privateStorage String 否 私有云CDN，上传的储存策略类型（ftp、oos、cos等等），支持字符串传入（英文逗号分隔，例如：ftp,oos）

注意：为防止图片恶意上传，Token为必须项，接口将认证该token，需要操作的图片必须在该token用户下，否则无法更新。

注意：publicCloud和privateStorage必须传入一项，需要操作的图片必须在该token用户下，否则无法更新。

响应参数

参数名称 类型 实例值 说明
code Integer 200 状态码，成功返回200，失败返回400
msg String success 提示信息
data Array|Object
