# 请求信息

功能 接口
请求方式 POST\GET
URL {:url('api/folderList')->suffix('')->domain(true)}

请求参数

参数名称 类型 是否必须 说明
token String 是 用户的秘钥
page int 否 分页（ 默认为 1 ）
limit int 否 限制条数（ 默认为 10 ）

响应参数

参数名称 类型 实例值 说明
code Number 200 状态码，成功返回200，失败返回400
msg String success 提示信息
data Array|Object 
"data":[
        {
            "id":1,
            "name":"jujingyi",
            "parentid":0,
            "createtime":1576062338
        }
    ]
 获取的文件夹信息数据

正确响应示例

{
    "code":200,
    "msg":"success",
    "data":[
        {
            "id":1,
            "name":"jujingyi",
            "parentid":0,

        }
    ]
}
